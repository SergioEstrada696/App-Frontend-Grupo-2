Proyecto de Empresa Metalurgica y Vidrios
Descripción

Este sistema permite gestionar de manera eficiente las operaciones principales de la empresa, incluyendo:

- Control de inventario de materiales (metal y vidrio)
- Gestión de productos y servicios
- Administración de clientes
- Registro de pedidos y ventas
- Control de reservas o solicitudes
- Visualización de reportes

Tecnologías Utilizadas

- ASP.NET Core MVC
- C#
- Entity Framework Core
- SQL Server
- HTML5, CSS3, JavaScript
